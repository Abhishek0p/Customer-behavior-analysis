# Customer Trends Data Analysis

This repository contains an end-to-end data analysis project focusing on customer shopping behavior. It explores customer trends, patterns, and insights through a combination of Python (Jupyter Notebook), SQL, and a Power BI dashboard.

## Project Structure

- `customer_shopping_behavior.csv`: The dataset used for the analysis.
- `Customer_Shopping_Behavior_Analysis.ipynb`: A Jupyter Notebook containing exploratory data analysis (EDA) and data processing steps.
- `customer_behavior_sql_queries.sql`: SQL queries used to extract and analyze key metrics from the data.
- `customer behavior dashboard.pbix`: A Power BI dashboard providing interactive visualizations of the customer trends.
- `Business Problem Document.pdf`: Documentation outlining the core business problems addressed in this analysis.
- `Customer Shopping Behavior Analysis.pdf` / `.pptx`: Presentation and detailed report of the findings.

## Highlights
- Conducted exploratory data analysis to understand purchasing habits and customer demographics.
- Wrote SQL queries to extract meaningful insights related to sales and customer behavior.
- Designed an interactive Power BI dashboard to visualize the trends and communicate actionable business insights.
